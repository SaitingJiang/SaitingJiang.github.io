vue 简介

Vue 是一套用于构建用户界面的前端框架。

使用 vue 构建用户界面，解决了 jQuery + 模板引擎 的诸多痛点，极大的提高了前端开发的效率和体验。

vue 全家桶：vue（核心库） vue-router（路由方案） vuex（状态管理方案） vue 组件库（快速搭建页面 UI 效果的方案）

vue 框架的特性，主要体现在如下两方面：① 数据驱动视图 ② 双向数据绑定

MVVM 是 vue 实现数据驱动视图和双向数据绑定的核心原理。它把每个 HTML 页面都拆分成了如下三个部分：

​ 在 MVVM 概念中：

​ View 表示当前页面所渲染的 DOM 结构。

​ Model 表示当前页面渲染时所依赖的数据源。

​ ViewModel 表示 vue 的实例，它是 MVVM 的核心。

​ 工作原理：

​ ViewModel 作为 MVVM 的核心，是它把当前页面的数据源（Model）和页面的结构（View）连接在了一起。

​ 当数据源发生变化时，会被 ViewModel 监听到，VM 会根据最新的数据源自动更新页面的结构

​ 当表单元素的值发生变化时，也会被 VM 监听到，VM 会把变化过后最新的值自动同步到 Model 数据源中

vue 的基本使用

① 导入 vue.js 的 script 脚本文件

② 在页面中声明一个将要被 vue 所控制的 DOM 区域

③ 创建 vm 实例对象（vue 实例对象）

<body>
   <!-- 2. 声明要被 vue 所控制的 DOM 区域 -->
   <div id="app">{{age}}</div>

   <!-- 1. 导入 vue 的脚本文件 -->
   <script src="./lib/vue-2.6.12.js"></script>

   <!-- 3. 创建 vue 的实例对象 -->
   <script>
     const vm = new Vue({//       -----------------------------------
       //                                                           -
       el: '#app',//    ------el指向的选择器，就是View试图区域        -
       //                                                           -
       data: {//        ------------                     new Vue()构造函数，得到的
         username: 'zs',//   data指向的对象                   vm实例对象就是ViewModel
         age: 20//          就是Model数据源                         -
       },//             ------------                               -
     })//                         -----------------------------------
   </script>
 </body>

​ vue 官方提供的 vue-devtools 调试工具，能够方便开发者对 vue 项目进行调试与开发

vue 的指令

指令（Directives）是 vue 为开发者提供的模板语法，用于辅助开发者渲染页面的基本结构。

1、内容渲染指令

v-text        {{ }}        v-html           内容渲染指令用来辅助开发者渲染 DOM 元素的文本内容。
<div id="app">
   <!-- <p v-text="username"></p> 
   <p v-text="gender">性别</p> -->
										//v-text 指令会覆盖元素内默认的值。
   <p>姓名：{{username}}</p>
   <p>性别：{{gender}}</p>
                      //相对于 v-text 指令来说，插值表达式在开发中更常用一些！因为它不会覆盖元素中默认的文本内容。	
   <p>---------</p>

   <p v-text="desc"></p>
   <p>{{desc}}</p>
   <p v-html="desc"></p>         //v-text 指令和插值表达式只能渲染纯文本内容。
 </div>
 <script>
   const vm = new Vue({
     el: '#app',
     data: {
       username: 'zs',
       gender: '男',
       desc: '<i style="color:red;">abc<i>'
     }
   })
 </script>

2、属性绑定指令

v-bind           为元素的属性动态绑定属性值
<input type="text" v-bind:placeholder="inputValue">      //简记为  :placeholder="inputValue"
<img v-bind:src="imgSrc" alt="">                         //        :src="imgSrc"
....
 data: {
          // 文本框的占位符内容
          inputValue: '请输入内容',
          // 图片的 src 地址
          imgSrc: './images/logo.png',
        },

在 vue 提供的模板渲染语法中，除了支持绑定简单的数据值之外，还支持 Javascript 表达式的运算

3、数据绑定指令

v-on （简记为@）              为 DOM 元素绑定事件监听
原生 DOM 对象有 onclick、oninput、onkeyup 等原生事件，替换为 vue 的事件绑定形式后，分别为：v-on:click、v-on:input、v-on:keyup
<h3>count 的值为：{{count}}</h3>
      <!-- TODO：点击按钮，让 data 中的 count 值自增 +1 -->
<button v-on:click="addCount">+1</button>            //等价于   @click="addCount"

通过 v-on 绑定的事件处理函数，需要在 methods 节点中进行声明

methods: {
          // 点击按钮，让 count 自增 +1
          addCount() {
            this.count += 1
          },

事件修饰符

.prevent 阻止默认行为 .stop 阻止事件冒泡 .once 绑定的事件只触发一次

<div class="inner" @click.once="onInnerClick">

按键修饰符

<input type="text" @keyup.enter="submit" @keyup.esc="clearInput" />

4、双向绑定指令

v-model      旨在不操作 DOM 的前提下，快速获取表单的数据（只能配合表单元素一起使用！）
<p>用户名是：{{username}}</p>
<input type="text" v-model="username" />
<select v-model="province">
        <option value="">请选择</option>
        <option value="1">北京</option>
        <option value="2">河北</option>
        <option value="3">黑龙江</option>
      </select>

修饰符

.number 自动将输入转换为数值型  .trim  自动过滤首尾留的空白字符   .lazy 在change时而不是input是更新

5、条件渲染指令

按需控制 DOM 的显示与隐藏

v-if           动态地创建或移除 DOM 元素，适合于运行时条件很少改变的场景
v-show         动态为元素添加或移除 style="display: none;" 样式，适合于频繁地切换的场景
<p v-if="type === 'A'">优秀</p>
     <p v-else-if="type === 'B'">良好</p>
     <p v-else-if="type === 'C'">一般</p>
     <p v-else>差</p>

6、列表渲染指令

v-for           者基于一个数组来循环渲染相似的 UI 结构
<ul>
      <li v-for="(user, index) in userlist" :key="user.id">
        <input type="checkbox" />
        姓名：{{user.name}}
      </li>
</ul>
....
data: {
          // 用户列表的数据
          list: [
            { id: 1, name: 'zs' },
            { id: 2, name: 'ls' },
          ],
        },
